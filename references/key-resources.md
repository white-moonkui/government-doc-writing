# 公文写作权威参考资料

> 本文件列出政务公文写作核心权威来源，供核实最新表述和政策依据时使用。

## 四、高频政策文件查询路径

1. 中央文件 → 中国政府网（www.gov.cn）→ "政策"频道
2. 党内法规 → 共产党员网（www.12371.cn）→ "党内法规"专栏
3. 最新政策 → 国务院客户端/省级政府客户端
4. 基层党建 → 共产党员网 → "基层党建"板块
