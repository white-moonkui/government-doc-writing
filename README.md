# government-doc-writing

> OpenCode Skill — 党政机关与国有企业政务、党务公文写作（资深"笔杆子"级）

## 简介

资深"笔杆子"级公文写作技能，涵盖法定公文（15种）、党务公文、领导讲话、工作报告、调研报告、述职报告、经验材料等全品类。基于 GB/T 9704-2012 格式标准，深度融合资深笔杆子的写作心法、逻辑框架、语言锤炼技巧、政治表述规范与常见错谬规避。

## 覆盖范围

### 法定公文（15种）
决议、决定、命令（令）、公报、公告、通告、意见、通知、通报、报告、请示、批复、议案、函、纪要

### 党务公文
- 党建工作报告、民主生活会材料、组织生活会材料
- 党课讲稿、党员发展材料、纪检监察文书
- 干部任免材料、主题教育材料

### 领导讲话（4种经典框架）
- 三段式、四个"聚力"式、肯定成绩+指出问题+部署任务、站位+形势+任务+保障

### 事务性公文
- 工作总结、工作计划、调研报告、述职报告
- 经验材料、会议方案、信息/简报

### 格式规范
- GB/T 9704-2012 标准速查（页面设置、字体字号、行距间距、版记要素）
- 高频易错格式提醒

### 政治表述规范
- 高频规范表述对照表（30+组）
- 重大提法规范（两个确立、两个维护、四个意识等）
- 2025-2026 最新时政术语

### 语言锤炼
- 句式变换、动词使用、起承转合
- "八股病"自我诊断清单
- 公文用语"四要四不要"

### 全流程 SOP
- 从领受任务到终审把关的6步写作流程

## 安装

### OpenCode（推荐）

```bash
# 方式1：通过 npx 安装（推荐）
npx skills add white-moonkui/government-doc-writing

# 方式2：直接复制到 OpenCode 技能目录
git clone https://github.com/white-moonkui/government-doc-writing.git
# 将 government-doc-writing 文件夹复制到 ~/.config/opencode/skills/
```

### 手动安装

将 `SKILL.md` 放入 opencode 技能目录：
- `~/.config/opencode/skill/opencode-skills/government-doc-writing/SKILL.md`

## 使用

在 OpenCode 对话中，当涉及以下场景时技能会自动触发：

| 场景 | 示例指令 |
|------|---------|
| 写公文 | "写一份通知/请示/报告/函" |
| 写领导讲话 | "写一篇年终总结大会讲话稿" |
| 写党务材料 | "写民主生活会对照检查材料" |
| 写总结报告 | "写一份季度工作总结" |
| 格式咨询 | "公文的标题字体是什么" |
| 文种选择 | "这个情况应该用请示还是报告" |
| 修改润色 | "帮我润色这段公文" |
| 审稿校对 | "帮我审核这份公文" |
