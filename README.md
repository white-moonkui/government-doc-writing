# government-doc-writing

> OpenCode 技能 — 党政机关与国有企业政务、党务公文写作

## 简介

资深"笔杆子"级公文写作技能，涵盖法定公文（15种）、党务公文、领导讲话、工作报告、调研报告等全品类。基于 GB/T 9704-2012《党政机关公文格式》国家标准，深度融合资深笔杆子的写作心法、逻辑框架、语言锤炼技巧、政治表述规范与常见错谬规避。

## 能力范围

- ✅ **15种法定公文** — 决议、决定、命令、公报、公告、通告、意见、通知、通报、报告、请示、批复、议案、函、纪要
- ✅ **党务公文** — 党建工作报告、民主生活会材料、组织生活会、党课讲稿、党员发展材料等
- ✅ **领导讲话** — 4种经典框架思路（三段式、聚力式、工作会式、大型会议式）
- ✅ **事务文书** — 工作总结、调研报告、述职报告、经验材料、会议方案、简报
- ✅ **格式规范** — GB/T 9704-2012 完整速查（页面设置、字体字号、行距、版记）
- ✅ **政治表述** — 高频正误对照、重大提法规范、时政术语
- ✅ **语言锤炼** — 句式变换、八股病诊断、起承转合、易错字词辨析
- ✅ **全流程 SOP** — 从领受任务到终审把关的6步写作流程

## 安装

### OpenCode

本技能已在 .config/opencode/skills/ 中，OpenCode 会自动发现。如需手动安装：

`ash
# 方法1：通过 npx 安装
npx skills add white-moonkui/government-doc-writing

# 方法2：直接复制到 OpenCode 配置目录
git clone https://github.com/white-moonkui/government-doc-writing.git
# 将 government-doc-writing 文件夹复制到 ~/.config/opencode/skills/
`

### 使用

当你在 OpenCode 中提出以下请求时，技能会自动触发：

- "写一份通知/请示/报告/函……"
- "写领导讲话稿/工作总结/调研报告"
- "帮我润色这份公文"
- "这个文种应该选什么？"
